# Meta Ad Campaign Creator

Create an ad. Follow this workflow:

1. **Ask which client** (if not in $ARGUMENTS)
2. **Ask the objective**: Engagement, Conversions, Awareness, Video Views, Traffic
3. **Ask the offer/message**: What are we promoting?
4. **Ask the budget**: Daily budget amount
5. **Ask the duration**: How many days to run

Then generate:

- **Ad headline** (short, punchy, scroll-stopping)
- **Primary text** (2-3 sentences max, human tone, drives action)
- **Description** (one line supporting the headline)
- **CTA button recommendation** (Book Now, Call Now, Learn More, etc.)
- **Audience targeting** (age, location radius, interests — based on client profile)
- **Suggested creative** (what image/video to use)
- **A/B test suggestion** (one alternative headline or text to test against)

If the user approves, use the meta-ads MCP to create the campaign directly in their ad account.

ALWAYS match the client's luxury brand identity. Ads must feel premium, not salesy.
