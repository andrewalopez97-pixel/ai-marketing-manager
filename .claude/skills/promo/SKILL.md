# Promotion Package Generator

Create a promotion. Follow this workflow:

1. **Ask which client** (if not in $ARGUMENTS)
2. **Ask the promotion type**: Seasonal, Gift Card, Limited-Time Offer, Free Add-On, Giveaway, Holiday
3. **Ask for details**: What service/offer, dates, discount amount, any restrictions

Then generate a COMPLETE promotion package:

- **Promo name** (catchy, on-brand)
- **Facebook post** (caption + hashtags + CTA)
- **Instagram post** (caption + hashtags + CTA)
- **TikTok script** (hook + 15-30 second video concept)
- **Ad copy** (for Meta ads — headline, primary text, description)
- **Poster copy** (headline, subtext, fine print for Canva design)
- **Landing page headline + subtext** (if applicable)

Everything must match the client's luxury identity from CLAUDE.md. Every piece must drive calls and bookings.
