# Trend Research

Research what's trending. Follow this workflow:

1. **Ask the focus** (if not in $ARGUMENTS): Nail designs, Colors, Services, TikTok trends, Seasonal, General beauty

Then use available tools:

- Use **brave-search** MCP to search for current trending nail styles, colors, viral content
- Search TikTok trends, Instagram reels trends, Pinterest trending

Generate:

- **Top 5 trends right now** (with brief description of each)
- **Content ideas per trend** (how to turn each trend into a post)
- **Which clients should use which trend** (match trend to client brand style)
- **Suggested hooks** (scroll-stopping first lines for each trend)
- **Hashtags to use** (trending + local SEO)

Keep it actionable — ideas to execute today, not a report to file away.
